SECTION 4: STEP-BY-STEP SERVICE SETUP
================================================================================

MONGODB ATLAS SETUP:
1. Go to mongodb.com/cloud
2. Create account (free tier)
3. Create M0 cluster (free forever tier)
4. Create database user with strong password
5. Add your IP to whitelist
6. Click "Connect" → "Connect Your Application"
7. Copy connection string
8. Replace <username> and <password> with actual values
9. Add to backend/.env as MONGODB_URI

REDIS CLOUD SETUP:
1. Go to redis.com/try-free
2. Create account (free tier available)
3. Create database
4. Copy Host, Port, Password
5. Add to backend/.env as:
   REDIS_HOST=host
   REDIS_PORT=port
   REDIS_PASSWORD=password

RESEND EMAIL SETUP:
1. Go to resend.com
2. Create account (100 emails/day free)
3. Add sending domain (must add DNS records)
4. Wait for DNS verification
5. Generate API key
6. Add to backend/.env:
   RESEND_API_KEY=re_xxxxx
   FROM_EMAIL=noreply@yourdomain.com

CLOUDINARY SETUP:
1. Go to cloudinary.com
2. Create account (25GB free tier)
3. Go to Settings → API Keys
4. Copy Cloud Name, API Key, API Secret
5. Add to backend/.env:
   CLOUDINARY_CLOUD_NAME=xxx
   CLOUDINARY_API_KEY=xxx
   CLOUDINARY_API_SECRET=xxx

ANTHROPIC SETUP:
1. Go to console.anthropic.com
2. Create account
3. Go to Settings → Billing → API Keys
4. Create new API key
5. Copy and add to backend/.env:
   ANTHROPIC_API_KEY=sk-ant-xxx

GENERATE JWT SECRETS:
Run in terminal:
  openssl rand -hex 16
Copy output twice (once for JWT_SECRET, once for JWT_REFRESH_SECRET)
